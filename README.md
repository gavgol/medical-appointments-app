# 💊 התורים שלי — Medical Appointments Tracker

אפליקציה פשוטה לניהול תורים רפואיים, מותאמת לעברית ולמשתמשים שאינם טכנולוגיים.

## תכונות
- ➕ הוספת תורים בקלות עם ויזארד צעד-אחר-צעד
- 🔄 תורים חוזרים (למשל כימותרפיה כל שבועיים)
- 🔔 התראות אוטומטיות — שבוע, 3 ימים, יום לפני, והיום
- ✏️ בחירת שעה מהירה או הקלדה ידנית
- 📱 ממשק RTL מלא בעברית
- 💾 שמירה מקומית בדפדפן

## התקנה

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel
3. Framework: Vite → auto-detected
4. Deploy!
